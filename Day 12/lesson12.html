#HTML

HTML

  • HTML ნიშნავს ჰიპერტექსტის მარკირების ენას
  • HTML არის სტანდარტული მარკირების ენა ვებ გვერდების შესაქმნელად
  • HTML აღწერს ვებ გვერდის სტრუქტურას HTML შედგება ელემენტების სერიისგან
  • HTML ელემენტები ეუბნებიან ბრაუზერს, თუ როგორ უნდა აჩვენოს შინაარსი
  • HTML ელემენტები ასახელებენ შინაარსის ნაწილებს, როგორიცაა "ეს არის სათაური", "ეს არის აბზაცი", "ეს არის ბმული" და ა.შ


#CSS

CSS

  • CSS ნიშნავს კასკადური სტილის ფურცლებს
  • CSS აღწერს, თუ როგორ უნდა იყოს ნაჩვენები HTML ელემენტები ეკრანზე, ქაღალდზე ან სხვა მედიაში
  • CSS დაზოგავს ბევრ სამუშაოს.მას შეუძლია ერთდროულად გააკონტროლოს მრავალი ვებ გვერდის განლაგება გარე სტილის ფურცლები ინახება CSS ფაილებში
